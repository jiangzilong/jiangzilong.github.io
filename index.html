面试要点梳理

概念

1、TCP三次握手四次挥手

2、HTTP/HTTPS

3、HTTP2.0/WebSocket

4、HTTP状态码

5、XSS CSRF

https://www.cnblogs.com/443855539-wind/p/6055816.html

XSS防御——(1) HTTPOnly response.setHeader(“Set-Cookie”, “cookiename=value;

Path=/;Domain=domainvalue;Max-Age=seconds;HTTPOnly”); (2) HTML实体转义

CSRF——(1) 验证码 (2) Referer Check (3) Token

使用Anti-CSRF Token 在URL中保持原参数不变，新增一个参数Token。Token的值是随机的（必须使用足够安全的随机数生成算法，或者采用真随机数生成器），其为用户与服务器所共同持有，可以放在用户的Session中，或者浏览器的Cookie中。 注意保密，尽量把Token放在表单中（构造一个隐藏的input元素），以POST提交，避免Token泄露

6、base64编码，编码表[A-Z, a-z, 0-9, +, /]，对应数字0-63，将二进制数据三个字节为一组进行分割，共3*8=24bit，再分为4组，每组6位，对应数字0-63，可根据数字在编码表中找到对应字符，完成编码。对于不足3位的，使用\x00补足，并在最后用1或者2个=表示补了几位

7、正向代理 反向代理
