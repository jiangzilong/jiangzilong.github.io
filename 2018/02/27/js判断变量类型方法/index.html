js判断变量类型方法

  • js

js中的基本变量类型有：

number、boolean、string、object、null、undefined、symbol

此外还可以区分出：array、function

通常来说都可以用typeof关键字来判断变量类型，首先测试不同类型分别返回什么内容：

可以看出number、boolean、string、function、undefined、symbol都可以有特定的返回值，但是array、object、null的返回值都为object，这就需要进行进一步判断

对于null

导致typeof null == 'object' 的原因是js中类型值存储占用32 bit，其中1~3位表示type tag，而object标记位是000，null作为空对象表示不指向任何数据所以都为0，这就造成了类型判断时也返回object的情况。

可以使用js特性null == undefined 来判断，如果一个变量null == undefined 返回true而typeof 返的又是object，那这个变量即为null

对于array

1、可以使用instanceof 关键字判断，如果obj instanceof Array 返回true，则为array类型；

2、一种更安全的方法，是使用Object原型中的toString方法，因为对于一个跨window对象的数组（例如使用iframe），在另一window环境中使用方法1会失效，这时可以调用Object.prototype.toString.call(obj) === '[object Array]' ，如果返回为true，则为array类型

余下的情况则为普通对象类型
