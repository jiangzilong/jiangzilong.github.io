数据双向绑定实现原理：

Modal=>View

改变数据 => 进入setter => 判断是否新值 => 是则对Dep实例调用notify()方法 => notify方法中通知订阅队列subs更新 => 遍历执行每个sub（Watcher实例）的update()方法 => update方法中设置绑定node的nodeValue完成View层的更新

View=>Modal

在observe阶段给对应元素添加input等监听事件，触发事件后更新Modal中的数据
